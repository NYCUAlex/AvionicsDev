%% This is a sounding rocket simulator for Asfoloth project
%% Several process to use this program
%% edited:2023/04/17 Alex Lin

step0: Open the Main.app
step1: Select model
step2: Modify the rocket properties(mass,CG) value
step3: For saving the properties, click the "File"->"save as"->"ver2023" or "ver2024" rocket, **Do not
 change the file name**.
step4: click "Flight simulation", modify the initail condition.
step5: click "Run 6DOF" button and see the graph in the right side.
